The Concept

How Syncurator Unites and Manages Your Media From Sale to Consumption

License Management

When digital content is sold online, the purchaser is effectively licensing content from a publisher. Publishers impose strict DRM software and bandwidth-hungry authentication methods that restrict the user experience and render media unusable. Syncurator captures license information and allows publishers to ensure the fair use of their content all while providing the user with the best experience possible.

User-Oriented Design

Syncurator works to serve licensed media to the user, taking away conventional barriers like platform DRM that impairs accessibility and detracts from the user experience. With Syncurator, the user is able to stream their own media and consume content on their own terms.

Top-Notch Security

Syncurator uses secure data storage practices and uses strong authentication to ensure content is only accessible to the content owner. Additionally, Syncurator uses transparent security measures so user experience is always the priority. Whether content is stored locally or being streamed, files are managed securely and quality is better than ever.

Platforms

Syncurator will be available for all platforms, with emphasis on utilizing open source codecs and technologies to allow access to Syncurator media on any device. This includes OpenSSL, open source application layer encryption technology, and the x265 codec. These protocols help make the product secure and fast, doing away with platform issues, restrictive DRM software, and insecure content protection solutions. The Project

Syncurator is composed of several key features that set it apart from other media management software platforms.

  • Cloud Facilitated

    Syncurator uses cloud technology to facilitate authentication, license management, and file storage across devices

  • Distributed Storage

    Syncurator uses an advanced interface to dynamically store media, optimizing storage space on devices to cache frequently used media. By reducing the need to stream media, user-generated data can be used to anticipate how and when media will be consumed.

  • Boundless Access

    By using persistent authentication mechanisms, Syncurator allows accessibility to your media, regardless of where you are and the quality of your connection. Syncurator also uses smart storage management to store media on the devices that best fit the user’s lifestyle.

  • License lending

    (In Progress)

    Syncurator is working with copyright holders to help users activate their physical licenses in a centralized, highly usable item. Users will no longer need to use highly restricted platforms like the iTunes Store, VUDU, CinemaNOW, or any of the solutions typically put forward by content creators. Syncurator is designed to put power back in the hands of users and ensure that content creators are able to be compensated properly. License lending will allow the transfer of content between individuals after a set amount of time, leading to more flexible access to content while protecting the profits of content creators in the spirit of the Constitution.
